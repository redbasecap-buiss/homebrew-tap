# Homebrew Tap

Custom Homebrew formulae by Nicola Spieser.

## Install

```bash
brew tap redbasecap-buiss/tap
brew install netscout
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| **netscout** | 10-in-1 network diagnostics toolkit (ping, dns, port scan, traceroute, HTTP probe, TLS check, speed test, WHOIS, LAN scan, TUI) |
